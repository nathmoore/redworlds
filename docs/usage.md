# Usage

The engine is under active development. Full usage examples will be added here as the
action functions (`apply_build`, `apply_swap`, `apply_reduce`) are implemented.

In the meantime, the best starting point is the examples directory:

- **[01_build_io_tables.ipynb](https://github.com/nathmoore/redworlds/blob/main/examples/01_build_io_tables.ipynb)** — how to load EXIOBASE and build baseline IO tables
- **[02_sample_payloads.ipynb](https://github.com/nathmoore/redworlds/blob/main/examples/02_sample_payloads.ipynb)** — sample BUILD/SWAP/REDUCE JSON payloads and result structure

See the [API Reference](api.md) for the full function signatures.
